GET /api/champions/list
Returns a JSON dictionary where the keys are champion names and the ids are champion ids

Examples

{"mega and.18":74,"tron bonne":2,"orchid":13,"vega":32,"medoi-san":10,"kamiccolo":70,"cyclops":52,"parasite":11, ...}

Supported Formats

json

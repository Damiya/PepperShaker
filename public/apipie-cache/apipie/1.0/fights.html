Fights
SaltyBet fights and their outcomes.

Supported Formats

json
>>>

GET /api/fights/:id
Looks up a given fight by id

Also see champions#show_fights_by_id, champions#show_wins_by_id, and champions#show_losses_by_id.

Supported Formats

json

Examples

{"id":1,"blue_champion_id":2,"red_champion_id":1,"bets_red":3752,"bets_blue":7079,"bet_count":44,"winner":2,"created_at":"2013-09-19T03:50:04.703Z","updated_at":"2013-09-19T03:50:04.703Z","match_id":4773}

Params

Param name Description
id
required

Fight id


Value: Must match regular expression /\d+/.
>>>

GET /api/fights/by_name/:champ_one/:champ_two
Looks up both champions and provides their stats in a single object

If the champions have fought each other before, includes a third subobject, -rematch-. See the second example

Supported Formats

json

Examples

{"left": [champion_one_object], "right": [champion_two_object]}
... ,"rematch":{"left_has_won":false,"right_has_won":true}}

Params

Param name Description
champ_one
required

Champion One name


Value: Must match regular expression /[^/]+/.
champ_two
required

Champion Two name


Value: Must match regular expression /[^/]+/.

• PepperShaker 1.0

PepperShaker is a site that automatically scrapes matches from SaltyBet.com and exposes the results via an API.

Resources

Champions
The various characters that can be used in fights.

Resource Description
GET /api/champions/:id Output the full champion object
GET /api/champions/:id/fights Output all fights the champ has participated in
GET /api/champions/:id/wins Output all fights the champion has won
GET /api/champions/:id/losses Output all fights the champion has lost
GET /api/c/:id Redirect to the champion on the FightMoney app (with comments and tags)
GET /api/champions/list Returns a JSON dictionary where the keys are champion names and the ids are champion ids

Fights
SaltyBet fights and their outcomes.

Resource Description
GET /api/fights/:id Looks up a given fight by id
GET /api/fights/by_name/:champ_one/:champ_two Looks up both champions and provides their stats in a single object
GET /api/fights/by_id/:champ_one/:champ_two Looks up both champions and provides their stats in a single object
GET /api/f/:champ_one/:champ_two Redirect to the matchup on the FightMoney app (with comments and tags)
